Artem Savchenko

⸻

🧠 Overview

This project is an interactive psychedelic audio visualiser and generative sound system built in Godot.

It combines:

* real-time audio playback
* generative visuals
* algorithmic sound patterns
* user-controlled parameters

The goal was to create something that feels less like a traditional instrument and more like a visual + audio experience / “trip” simulation.

⸻

🎛 Features

🔊 Audio System

* Polyphonic sound playback (multiple sounds at once)
* Keyboard-controlled sound triggering
* Adjustable:
    * Volume
    * Pitch
* Random pattern generator (auto-play mode)

⸻

🌈 Visual System

Fractal Visuals

* Multiple animated fractal systems
* Independent movement and rotation speeds
* Dynamic shape distortion
* Audio-reactive pulsing

Blob System

* Organic, fluid shapes generated per sound
* Randomised scale, rotation and transparency
* Creates a “liquid / melting” visual effect

Particle System

* Floating particles with dynamic connections
* Movement across the screen
* Adjustable speed and density

Shader Effects

* Screen distortion using a custom shader
* Color shifting and wave-based UV manipulation
* Creates a psychedelic “melting screen” effect

⸻

🎚 Control Panel

The UI is designed as a live control panel allowing real-time manipulation:

Sound Controls

* Volume
* Pitch
* Random playback speed

Fractal Controls

* Number of fractals
* Movement speed
* Rotation speed

Particle Controls

* Particle count
* Particle speed

Blob Controls

* Number of blobs generated per sound

Playback

* Start / Stop random generative sequence

⸻

⚙️ Computational Elements

This project includes several computational systems:

* Algorithmic sound generation (random sequencing)
* Procedural visual generation (fractals, blobs, particles)
* Real-time parameter control via UI
* Signal-based interaction between UI and systems
* Use of multiple nodes and modular scripts
* Time-based animation and interpolation

⸻

🧪 Development Process

The project was developed iteratively:

1. Built a basic sound system with keyboard input
2. Added visual feedback (simple shapes)
3. Introduced shaders for distortion effects
4. Replaced simple visuals with:
    * fractal systems
    * organic blobs
    * particle connections
5. Implemented a control panel for real-time interaction
6. Added a generative music system

The focus shifted from a simple soundboard to a fully interactive audiovisual environment.
